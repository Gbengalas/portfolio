About Me

Freelance Web Designer & Developer based in Lagos, Nigeria — building fast, functional WordPress sites and web apps that help small businesses grow.

Custom WordPress sites & web apps

I'm a self-taught developer with roughly three years of hands-on experience, specializing in WordPress, WooCommerce, and PHP-driven builds — plus a growing practice in React for more interactive interfaces. Before development, I worked in sales and marketing, which shapes how I think about conversion, clarity, and what actually makes a site work for a business.

My core stack is WordPress, WooCommerce, PHP, JavaScript, TypeScript, HTML/CSS, Bootstrap, MySQL, Git and REST API integration. I've delivered projects across e-commerce, logistics, and education — each shipped with a measurable outcome, not just a finished template.
